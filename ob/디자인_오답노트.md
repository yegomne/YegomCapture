# 📝 디자인 오답 노트 (Lessons Learned)
*작성일: 2026-04-20*

### 1. 프리미엄 무드에서는 빈 깡통(Placeholders) 금지!
레이아웃을 구성하거나 구조를 잡을 때, 이미지 자리에 빈 SVG 스켈레톤이나 Placeholder를 남겨두면 프리미엄 무드가 뚝 떨어집니다. 사이트에 들어갈 이미지가 비어 있다면, **기획 의도에 맞는 고화질 에셋을 AI 렌더링으로 즉각 생성**하여 부착해야 합니다. (절대 빈 공간으로 방치하지 말 것!)

### 2. 무한 롤링 마퀴(Marquee) 속도 및 가독성 최적균형
화면을 가로지르는 텍스트 배너는 **속도가 너무 빠르면 사용자 인지에 방해**가 됩니다. (ex. `20s` -> `40s` 이상으로 여유롭게 세팅). 또한 텍스트 사이즈는 `text-lg` 이상으로 시원시원하게 키워야 가독성과 세련미를 모두 잡을 수 있습니다.

### 3. 레이아웃 CSS 변경 시 부모 컨테이너(Grid/Flex) 파괴 주의
이커머스/제품 소개 영역의 좌우 분할 레이아웃을 수정할 때, 무심코 자식 요소의 display 속성을 건드려 부모의 `flex`나 `grid` 구조가 수직으로 찌그러지는(깨지는) 일이 없도록 클래스를 보수적으로 확인 후 수정합니다.
